# Weather Alerts App
A simple JavaScript web application that fetches active weather alerts by U.S. state using the National Weather Service API. Users enter a state code, click the fetch button, and view current alerts or error messages.

## Table of Contents
- [Installation](#installation)

## Installation
1. Clone the repository:
git clone git@github.com:safiabulle/course-3-module-4-external-apis-and-fetching-lab.git

2. Open the project folder:
cd course-3-module-4-external-apis-and-fetching-lab

3. Open index.html in your browser.

## Usage
Enter a U.S. state code (example: CA, TX, FL).
Click the Fetch Alerts button.
View active weather alerts displayed on the page.
